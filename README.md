# Python共享自习室管理系统 python1044

## 项目概述


## 技术栈

- 后端：Python + Django
- 前端：Vue
- 数据库：MySQL

## 功能模块

系统主要包括以下功能模块：

- 用户管理
- 自习室资源管理
- 时间预约管理
- 系统权限控制

## 系统角色

- 管理员：负责系统的日常维护和用户管理。
- 用户：可进行自习室资源的预约和使用。

## 运行环境

- 后端：需安装Python及Django环境。
- 前端：需安装Node.js版本14以上，以及cnpm环境。

## 部署步骤

1. 安装后端所需Python及Django环境。
2. 安装前端所需Node.js和cnpm环境。
3. 配置数据库MySQL。
4. 部署前后端项目。

## 目录结构

```
- backend             # 后端代码目录
- frontend            # 前端代码目录
- docs                # 项目文档
- resources           # 项目资源文件
```
